Projects

Bean Scene App

This Android app that was built using React / React Native, Node.js, Express and MongoDB is for the Bean Scene Restaurant that allows their managers and other staff to create and manage customers' reservations and orders, and all menu items to be available for ordering.

Repos: https://github.com/Bean-Scene-Zachary-and-Alex/bean-scene-app-zach-alex and https://github.com/Bean-Scene-Zachary-and-Alex/Bean-Scene-Express

Bean Scene Reservations Website

This website that was built using C#, .NET Core, Entity Framework, MVC, Bootstrap, HTML, CSS and JavaScript is for the Bean Scene Restaurant that has the same purposes as the Bean Scen App. This website also allows their customers to book a reservation and / or view menus that are offered by Bean Scene.

Repo: https://github.com/Bean-Scene-Zachary-and-Alex/Bean-Scene-ReservationSystem
